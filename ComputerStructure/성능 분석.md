성능 분석을 위해선느 무엇을 기준으로 측정할지 결정해야한다.

    - 이벤트 발생 횟수
    - 결과 도출에 걸리는 시간
    - 환경변수 크기

MIPS : 처리량 또는 실행 속도의 척도가 되는 단위 시간당 수행되는 계산량
Million Instruction Per Second : n / (t x 10^6)

벤치마크 프로그램 : 성능 평가를 위해 개발된 표준 프로그램

프로그램 실행에 필요한 시간 = (프로그램 실행에 필요한 명령어 총 개수) x CPI x T
CPI(Cycles Per Instruction) : 모든 명령을 실행하는데 필요한 프로세서 클록 수의 가중 평균 = Σ (명령어 비율)x(필요 클럭 수)
T : 프로세서 클록 사이클 한 주기

CPU Sim : CPU 세부 사항을 직접 지정해서 시뮬레이션할 수 있게 메모리와 레지스터, 마이크로 명령, 기계 명령 세트와 어셈블리어 명령 등을 포함한 프로그램 패키지. 기계어나 어셈블리어 프로그램을 작성하여 CPU를 실행할 수 있다.
기본적으로 레지스터 전송 수준에서 시뮬레이션.
